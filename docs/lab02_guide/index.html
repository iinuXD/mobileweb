แนะนำสถานที่น่าสนใจในมหาวิทยาลัยขอนแก่น

รหัส: 663380201-3 | ชื่อ: นายกิตติพัฒน์ ราชศรีเมือง

ยินดีต้อนรับสู่รั้วมอดินแดง

มหาวิทยาลัยขอนแก่นมีพื้นที่กว้างขวางและมีสถานที่สวยงามมากมาย นี่คือ 4 สถานที่แนะนำที่คุณไม่ควรพลาด

1. ศาลเจ้าพ่อมอดินแดง

สถานที่ศักดิ์สิทธิ์อันเป็นที่เคารพสักการะของชาว มข. เป็นศูนย์รวมจิตใจของนักศึกษาและบุคลากร

2. อุทยานเทคโนโลยีการเกษตร มหาวิทยาลัยขอนแก่น

พื้นที่ศูนย์กลางวิจัยและพัฒนาด้านเทคโนโลยีการเกษตร แหล่งเรียนรู้ที่ทันสมัยและสวยงาม

3. บึงสีฐาน

สถานที่พักผ่อนหย่อนใจยอดฮิต เหมาะแก่การวิ่งออกกำลังกายและชมพระอาทิตย์ตกดิน

4. หอสมุดกลาง

แหล่งเรียนรู้ที่ทันสมัย ดีไซน์สวยงาม เปิดบริการนั่งอ่านในพื้นที่ทุกวัน วันจันทร์-ศุกร์ เวลา 08.30-20.00 น. วันหยุด เวลา 09.00-17.00 น.
